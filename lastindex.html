Expected File Format

Your file should include the following columns:

  • user_id - User identifier
  • transaction_date - Date and time of transaction
  • type - Transaction type (PAYMENT, TRANSFER, CASH_OUT, etc.)
  • amount - Transaction amount
  • old_balance - Old balance before transaction
  • new_balance - New balance after transaction
  • source_account - Source account number
  • destination_account - Destination account number
  • branch - Branch location
  • currency - Currency type
  • device - Device used for transaction
  • ip - IP address
  • location - Geographic location

All columns are required for accurate fraud detection.
